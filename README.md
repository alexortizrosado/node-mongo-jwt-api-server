# node-mongo-jwt-api-server
Node.js + MongoDB API with JWT authentication

##  Getting Started

1. Set up local mongodb

        brew tap mongodb/tap
        brew services start mongodb-community@4.4
        brew services start mongodb-community@4.4

2. Instal dependencies

        npm install


3. Set up .env
    
        cp .env.example .env

4. Start server

        npm start
